# Amazon Revenue Prediction System

## Problem Statement
Predict product revenue based on pricing, category, region and payment behavior.

## Dataset
- 19 features
- Target: Revenue
- Preprocessing: Encoding, feature engineering

## Model
- Random Forest Regressor
- Hyperparameter tuning using GridSearchCV

## Performance
- MAE: 266
- RMSE: 359
- R² Score: 0.52

## Deployment
- Built using Streamlit
- User inputs 19 features
- Returns predicted revenue
